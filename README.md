# 🏡 Imobiliária Hill

Uma aplicação web moderna para apresentação de imóveis, desenvolvida com foco em experiência do usuário, design responsivo e navegação intuitiva.

## 📖 Sobre o Projeto

A Imobiliária Hill é uma plataforma web criada para facilitar a divulgação de imóveis residenciais e comerciais, permitindo que potenciais clientes encontrem
propriedades de forma rápida e organizada.

O projeto foi desenvolvido utilizando tecnologias front-end modernas, priorizando desempenho, usabilidade e uma interface visual atrativa.

## ✨ Funcionalidades

- 🏠 Listagem de imóveis disponíveis
- 🔍 Navegação intuitiva entre propriedades
- 📱 Layout responsivo para todos os dispositivos
- 📸 Galeria de imagens dos imóveis
- 📞 Área de contato para atendimento
- 🎨 Interface moderna e elegante

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Design Responsivo
